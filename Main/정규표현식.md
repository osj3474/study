# 정규표현식

```javascript
^x            # x로 시작하는 문자열
x$            # x로 끝나는 문자열
.             # 임의의 문자

\d            # 숫자
\D            # \d가 아닌 것
\w            # 문자(알파벳, 숫자, _)
\W            # \w가 아닌 것
\s            # 공백 문자
\S            # \s가 아닌 것

[]            # [] 중에 하나
[xy]          # x or y 하나
[^xy]         # x or y 제외한 하나
[0-5]         # 0~5 중에 하나

{}            # {}개
x{3}          # x 3개
x{2,5}        # x 2개 이상 5개 이하

x+            # x 한번 이상
x*            # x 0번 이상
x?            # x 있어도 되고, 없어도 되고

()            # () 추출할 수 있음
(x|y)         # x문자열 or y문자열 중에 하나

r"x"          # x를 raw string으로 봄 ex) r"\n"은 \, n이 각각 문자열로 취급됨
```









# 사용

일단, 큰 틀!

모듈 가져오고,

```python
import re
```



그리고 찾을 문자열에 대한 정규표현식을 정의

```python
p = re.complie(정규표현식)
x_lst = re.findall(p, "적용할 문자열")
```



적용 문자열에서 특정 부분을 추출하고 싶을 때는, ()를 사용한다.

```python
txt = "number: 15, number: 22"
p = re.compile("number:\s(\d+)")
x = re.findall(p, txt)
print(x)                           # 결과 : ['15', '22']
```





# 실습예제

1. 특정 단어(h)로 시작하는 단어 추출

   ```python
   import re
   
   txt = "Hanst of handong is holy."
   p = re.compile("\s*([Hh]\w+)")
   x = re.findall(p, txt)
   print(x)
   ```

   ##### [결과]

   ```powershell
   ['Hanst', 'handong', 'holy']
   ```

   



2. 로그 데이터에서 날짜 추출해서 Dataframe으로 만들기

   ```python
   import re
   import pandas as pd
   
   date_lst = list()
   file = './input.txt'
   with open(file, 'r') as fp:
       while True:
           f = fp.readline()
           if f == '':break
           p = re.compile(".*\"\$date\":\"(\d{4})-(\d{2})-(\d{2}).*")
           date_lst.append(list(re.findall(p, f)[0]))
   
   df = pd.DataFrame(date_lst,columns=['Year', 'Month', 'Day'])
   print(df)
   ```

   ##### [결과]

   ```powershell
   9  2020    06  12
   ```
