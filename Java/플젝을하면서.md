7. Json으로 응답할 때, LocalDateTime을 제대로 응답하려면, 해당 컬럼 변수에 

   ```java
   @JsonFormat(pattern = "yyyy-MM-dd HH:mm:ss", timezone = "Asia/Seoul")
   ```

   를 넣자.

8. REST API 는

   - C : @PostMapping
   - R : @GetMapping
   - U : @PutMapping
   - D : @DeleteMapping

   참고로, delete는 실제 DB에서 값을 지우는게 아니라, deleteDt에 delete시간을 넣고,

   ```
   findByIdAndDeleteDtIsNull 
   ```

   로 조회를 하자.
